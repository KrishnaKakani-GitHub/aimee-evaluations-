aimee-evaluations: 
Evaluation framework for Aimee, a real-time multimodal AI healthcare assistant. Captures and analyzes voice-agent conversation transcripts to assess conversational accuracy, contextual reasoning, and workflow performance. Supports an iterative conversation → evaluation → refinement loop across 150+ structured test scenarios, documenting edge cases and driving continuous improvement of agent behavior in HIPAA-aligned clinical environments.

Evaluation Data: 
Each evaluation record captures a transcript, the scenario being tested, a qualitative assessment, and a pass/partial/fail score. Scenarios span multilingual handling, symptom triage, scope boundaries, transparency, and ASR edge cases.
